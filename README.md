
# vim-pandoc-preview

Live preview markdown files in the browser while editing.

# Requirements

Follow links for installation instructions:

- [pandoc](https://github.com/jgm/pandoc)
- [live-server](https://github.com/tapio/live-server)

# Usage

```vim
" Start preview
:PandocPreview
:PPreview

" Stop preview
:PandocPreviewStop
:PPreviewStop
```
